About

Sticky is a web application which lets you add notes for multiple purposes (Work tasks, Daily activities, Study projects ... ). Built with MEAN stack (MongoDB, ExpressJS, AngularJS, and Node.js).

Pre-requisites
node.js - Download page
Angular CLI - Download page
MongoDB - Download page
Installation - Frontend

cd into frontend folder and run:

npm install
Start the angular application

run:

ng serve
Installation - Backend

cd into backend folder and run:

npm install
Start the express server

run:

npm run start:server
